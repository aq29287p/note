# 136.  
## 解答
1. a^a = 0 , a^0=a   

### 實作

:::: code-group
::: code-group-item c++

``` cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        //bc
        if(nums.size()==0) return -1;
        int sum=0;
        for(int i=0; i<nums.size();i++){
            sum=sum^nums[i];
        }
        return sum;
    }   
};
```

:::
::: code-group-item python

``` python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        #bc
        if not nums: return -1
        
        res=0
        for i in nums:
            res=res^i
        return res
```
